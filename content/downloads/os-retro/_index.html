DeployKit

DeployKits are live environments, bootable from a removable storage devices such as floppy disks and USB sticks or optical media such as CDs and DVDs. These releases contain minimal environments for running an AOSC OS installer to guide you through system installation. We recommend using DeployKit for system installation where possible.

Tarballs and Images

Download individual tarballs and images for manual deployment. Tarballs contain full system root, which you could use to decompress and configure as a bootable system or a chroot environment. Images are often used for ARM-based devices which uses removable memory cards as boot media.
