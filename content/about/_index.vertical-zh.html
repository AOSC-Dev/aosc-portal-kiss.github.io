社群歷史

安同開源社區 (AOSC) 的最初形態是由白銘驄與他的兩位同學 於 2011 年末創立的安同開發團隊 (Anthon.Dev)。開發團隊因組織 管理不善於一年內夭折，而後於 2012 年末重組為開源社區並 持續至今。

我社的主要項目 AOSC OS 最初名為「安同 OS (AnthonOS)」，一款基於 現已棄用的 SUSE Studio 製作的 openSUSE 衍生發行版。一年後又 轉而基於 Debian 衍生開發，著重於 CJK（中文、日文及韓文）支援 並基於 KDE 4 設計了一套自訂使用者界面。

2014 年，得益於原東方之心 Linux 開發者 Icenowy Zheng 的幫助， 社區以 Anthon :Next 為名立項，將安同 OS 轉為獨立構建。 安同 OS 獨立後更名為 AOSC OS，即我們社群目前維護的通用 Linux 發行版。

隨後幾年，維護者們不斷擴充 AOSC OS 的套件庫體積，並引入了 多架構支持。如今，得力於開發者們的持續貢獻，AOSC OS 的 軟硬體支援仍在不斷擴充壯大。
